🛒 Shopping Cart

Transform Shopping into Seamless, Effortless Experiences

📖 Overview

Shopping Cart is a full-stack e-commerce web application built using Spring Boot, Java, HTML, CSS, and JavaScript. It provides a scalable and structured platform to build modern online shopping experiences with ease. The project follows a clean architecture, making it easy to extend and maintain.

🚀 Key Features
🎨 Modular UI Pages – Home, product listings, login/signup, and admin dashboard
🔧 Admin Panel – Add, update, and manage products & categories
🔍 Smart Search & Filters – Improve user shopping experience
🔐 Authentication System – Secure user login & registration
🏗️ Spring Boot Backend – Scalable and production-ready architecture
📐 Clean Code Structure – MVC pattern (Controller, Service, Repository)
🛠️ Tech Stack
Backend: Spring Boot, Java
Frontend: HTML5, CSS3, JavaScript
Build Tool: Maven
🚀 Getting Started
✅ Prerequisites
Java (JDK 8 or above)
Maven
IDE (IntelliJ / Eclipse)
⚙️ Installation
git clone https://github.com/abhishekprojects/Shopping_Cart
cd Shopping_Cart
mvn install
🎯 Run the Application
mvn exec:java
🧪 Testing
mvn test
